STEP 1: GitHub Setup
┌─────────────────────────────────────────────────────────────┐
│ 1. Go to https://github.com/new                            │
│                                                             │
│ 2. Create new repository:                                  │
│    Repository name: petra-ai-dashboard                     │
│    Description: (optional)                                 │
│    Public: YES                                             │
│                                                             │
│ 3. Copy GitHub URL from setup page                         │
│    Format: https://github.com/YOUR_USERNAME/petra...      │
│                                                             │
│ 4. Open Command Prompt:                                    │
│    Win+R → type: cmd → press Enter                         │
│                                                             │
│ 5. Navigate to project:                                    │
│    cd "c:\Users\austi\Videos\ai dashboard"                 │
│                                                             │
│ 6. Run these commands (one by one):                        │
│                                                             │
│    git init                                                │
│                                                             │
│    git add .                                               │
│                                                             │
│    git commit -m "Petra AI Dashboard - Ready"              │
│                                                             │
│    git branch -M main                                      │
│                                                             │
│    git remote add origin [YOUR_GITHUB_URL]                 │
│    (Replace [YOUR_GITHUB_URL] with your URL from step 3)  │
│                                                             │
│    git push -u origin main                                 │
│                                                             │
│ ✅ Done! Code is on GitHub                                │
└─────────────────────────────────────────────────────────────┘

STEP 2: Vercel Deployment
┌─────────────────────────────────────────────────────────────┐
│ 1. Go to https://vercel.com                                │
│                                                             │
│ 2. Click "Sign Up"                                         │
│    Choose: "Continue with GitHub"                          │
│    Authorize Vercel to access GitHub                       │
│                                                             │
│ 3. Click "New Project"                                     │
│                                                             │
│ 4. Click "Import Git Repository"                           │
│                                                             │
│ 5. Find and click: petra-ai-dashboard                      │
│                                                             │
│ 6. Vercel auto-detects settings (should be correct):       │
│    Framework: Node.js ✓                                    │
│    Root Dir:  ./ ✓                                         │
│                                                             │
│ 7. Click "Environment Variables"                           │
│                                                             │
│ 8. Add environment variable:                               │
│    Name:  OPENROUTER_API_KEY                               │
│    Value: [your API key from openrouter.ai]                │
│    (Can skip this for now, chat won't work until added)    │
│                                                             │
│ 9. Click "Deploy"                                          │
│                                                             │
│ 10. WAIT 2-3 MINUTES                                       │
│     You'll see: Building... → Validating... → Ready ✅     │
│                                                             │
│ ✅ Done! Your app is deployed                             │
└─────────────────────────────────────────────────────────────┘

⚠️  TROUBLESHOOTING
═════════════════════════════════════════════════════════════════

404 Error After Deployment:
  → Vercel → Deployments → Select latest
  → Click "Redeploy"
  → Wait 2-3 minutes

Chat Not Responding:
  → Add OPENROUTER_API_KEY to Vercel environment variables
  → Get key at: https://openrouter.ai
  → Redeploy from Vercel dashboard

Git Commands Failed:
  → Make sure you're in correct folder
  → Check GitHub URL is correct
  → Try: git remote -v (to verify remote URL)

Build Failed:
  → Check Vercel logs: Deployments → Select latest → Logs
  → Common issues listed in FIX_VERCEL_404.md
