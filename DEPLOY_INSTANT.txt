🚀 DEPLOY NOW - 3 MINUTE DEPLOYMENT GUIDE
═════════════════════════════════════════════════════════════════

Your project is 100% ready. Deploy in 3 minutes!


⏱️  STEP 1: GitHub Setup (2 minutes)
─────────────────────────────────────────────────────────────────

1. Go to https://github.com (create account if needed)

2. Click [+] > New repository

3. Name it: "petra-ai-dashboard"

4. Copy the commands from GitHub (should look like below)

5. Open Command Prompt in your project folder:
   cd "c:\Users\austi\Videos\ai dashboard"

6. Run these commands (one at a time):
   
   git init
   git add .
   git commit -m "Initial Petra AI Dashboard"
   git branch -M main
   git remote add origin https://github.com/YOUR_USERNAME/petra-ai-dashboard.git
   git push -u origin main

   NOTE: Replace YOUR_USERNAME with your GitHub username!


⏱️  STEP 2: Vercel Deployment (1 minute)
─────────────────────────────────────────────────────────────────

1. Go to https://vercel.com

2. Click "Sign Up" (use GitHub login)

3. Click "New Project"

4. Click "Import Git Repository"

5. Select your "petra-ai-dashboard" repo

6. Click "Import"

7. Vercel auto-detects the configuration
   - Framework: Node.js
   - Root Directory: ./

8. Click "Environment Variables"

9. Add this variable:
   Name: OPENROUTER_API_KEY
   Value: [Get from https://openrouter.ai]
   
   (If you don't have an API key yet, you can add it later)

10. Click "Deploy"

11. Wait 2-3 minutes...


⏱️  STEP 3: Verify (< 1 minute)
─────────────────────────────────────────────────────────────────

1. When deployment finishes (Vercel shows "Ready"):
   
2. Copy the provided URL (like petra-ai-dashboard.vercel.app)

3. Open in browser: https://your-project.vercel.app

4. See the dashboard load? ✅ DEPLOYED!

5. Test chat:
   - Type "hello" in chat box
   - Press Enter
   - Should get AI response (if API key is set)


✅ EXPECTED RESULT:
─────────────────────────────────────────────────────────────────

You should see:
✅ Dashboard with chat interface
✅ Activities list (5 items)
✅ Announcements (3 items)
✅ Dark/Light theme toggle
✅ Chat working (sends and receives messages)


🔑 IF YOU DON'T HAVE OPENROUTER API KEY:
─────────────────────────────────────────────────────────────────

1. Go to https://openrouter.ai

2. Click "Sign Up"

3. Create account (email or social login)

4. Go to API Keys

5. Copy your API key

6. Go back to Vercel:
   - Project → Settings → Environment Variables
   - Add: OPENROUTER_API_KEY = [your key]
   - Redeploy

❓ TROUBLESHOOTING:
─────────────────────────────────────────────────────────────────

Q: Chat returns error
A: Add OPENROUTER_API_KEY environment variable to Vercel.

Q: Still not working
A: Check Vercel Logs (Deployments → Select Latest → Logs)
